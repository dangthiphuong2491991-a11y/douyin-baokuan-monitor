# 爆款监控 · 抖音博主更新雷达

一个 Windows 桌面软件：监控抖音博主更新、发现库内爆款、在线播放、无水印下载。

## 功能
- **博主监控**：粘贴博主主页链接（或分享口令 / 视频链接自动识别作者）加入监控，定时检查更新并弹窗提醒。
- **更新动态**：按博主分类，显示每个博主更新了几条，点进去看明细，可在线播放或手动下载。
- **发现爆款**：
  - 「我的库」：在你监控的博主里，按时间窗 + 点赞排序取前 N。
  - 「相关推荐」：用一条种子视频，从相关推荐里挖同题材的其他博主。
- **在线播放**：任意视频点 ▶ 直接在软件里播放（后端带 Referer 代理，支持拖动进度条）。
- **下载库**：无水印原片按博主名分类保存，位置可自定义。
- **抖音登录**：扫码 / 从浏览器导入 / 粘贴 Cookie。登录后每个博主能翻页拿到更多作品。
- **导出/导入博主**：换电脑时一键搬走博主列表（不含登录信息）。

> 注意：抖音在作品接口层面**不返回真实播放量**，所有"够火"的排序均按**点赞量**。

## 运行（源码）
```bash
pip install -r requirements.txt
python desktop.py        # 桌面窗口
# 或 python app.py       # 仅后端 + 浏览器打开 http://127.0.0.1:8790
```

## 打包成 exe（分发给他人）
```bash
pyinstaller build.spec
# 产物在 dist/ 目录
```

## 数据与隐私
- 登录 cookie、博主配置存在本地 `data/`，**不会**上传。
- 下载的视频存在 `downloads/`（或你自定义的位置）。
